# ⌘ Zed macOS Cheatsheet

## General & Application
| Shortcut | Action |
| :--- | :--- |
| **⌘ ,** | Open Settings |
| **⌘ ⇧ P** | **Command Palette** (Run any command) |
| **⌘ ⇧ X** | Open Extensions Store |
| **⌘ Q** | Quit Zed |
| **⌘ ⇧ C** | Open Collaboration Panel |

## File & Project
| Shortcut | Action |
| :--- | :--- |
| **⌘ N** | New File |
| **⌘ ⇧ N** | New Window |
| **⌘ O** | Open File / Folder |
| **⌘ S** | Save |
| **⌘ ⇧ S** | Save As... |
| **⌘ ⌥ S** | Save All |
| **⌘ W** | Close Tab / Buffer |
| **⌘ ⇧ W** | Close Window |

## Navigation (Go)
| Shortcut | Action |
| :--- | :--- |
| **⌘ P** | **Go to File** (Fuzzy Finder) |
| **⌘ ⇧ O** | Go to Symbol (in current file) |
| **⌘ T** | Go to Symbol (in project) |
| **⌃ G** | Go to Line |
| **F12** | Go to Definition |
| **⌘ ⇧ F** | Find in Project |
| **⌘ F** | Find in File |
| **⌘ ⌥ -** | Go Back (Previous cursor position) |
| **⌘ ⇧ -** | Go Forward |

## Editing
| Shortcut | Action |
| :--- | :--- |
| **⌘ Z** | Undo |
| **⌘ ⇧ Z** | Redo |
| **⌘ /** | Toggle Line Comment |
| **⌘ [** / **]** | Indent / Outdent |
| **⌘ ⌃ ↑** | Move Line Up |
| **⌘ ⌃ ↓** | Move Line Down |
| **⌘ ⏎** | Insert line after |
| **⌘ ⇧ ⏎** | Insert line before |
| **⌘ .** | Code Actions / Quick Fix |

## Selection & Multi-Cursor
| Shortcut | Action |
| :--- | :--- |
| **⌘ D** | **Add Selection to Next Occurrence** |
| **⌘ ⌃ D** | Select *All* Occurrences |
| **⌘ L** | Select Current Line |
| **⌥ ↑** | **Expand Selection** (Syntactic / Smart Select) |
| **⌥ ↓** | Shrink Selection |
| **⌥ ⇧ Drag** | Column / Box Selection (Mouse) |
| **⌥ Click** | Add Manual Cursor |

## View & Layout
| Shortcut | Action |
| :--- | :--- |
| **⌘ B** | Toggle Left Dock (File Tree) |
| **⌘ J** | Toggle Bottom Dock |
| **⌘ ⌥ B** | Toggle Right Dock |
| **⌃ `** | Toggle Terminal Panel |
| **⌘ K →** | Split Pane Right |
| **⌘ K ↓** | Split Pane Down |
| **⌘ K ←** | Split Pane Left |
| **⌘ K ↑** | Split Pane Up |
| **⌘ +** / **-** | Zoom In / Out |

---
* **⌘** Command / **⇧** Shift / **⌥** Option / **⌃** Control / **⏎** Return
